# Loop prototype

Vite React prototype for Loop, integrated for Vercel testing.

## What is included

- Latest Loop prototype design and logo
- Latest course master database converted into `src/courseData.js`
- 5 courses, with tee-level par, yardage and stroke index
- Loop insight engine wired into review, next-round plan, smart prompt and practice screens
- Practice spelling standardised
- Save hole and continue button sizing retained

## Local setup

```bash
npm install
npm run dev
```

## Vercel

- Framework preset: Vite
- Build command: `npm run build`
- Output directory: `dist`
- Install command: `npm install`
