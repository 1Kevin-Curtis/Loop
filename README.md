# Loop prototype

Vite React prototype for Loop with:

- Three verified courses wired into the round capture flow
- White, Yellow and Red tee selection
- Dynamic par, yardage and stroke index
- Updated Loop logo asset
- Practice spelling standardised
- Fixed Save hole and continue button sizing

## Local setup

```bash
npm install
npm run dev
```

## Vercel

Framework preset: Vite
Build command: npm run build
Output directory: dist
Install command: npm install
