# Loop three-course prototype

React/Vite prototype for Loop with three verified courses wired into the round setup flow.

## Run locally

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Create a new GitHub repo.
2. Upload all files in this folder.
3. In Vercel, import the repo.
4. Use the default Vite settings:
   - Build command: `npm run build`
   - Output directory: `dist`
